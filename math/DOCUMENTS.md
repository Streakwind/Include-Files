# DOCUMENTATION

Quick guide for `example.h`. <br> <br>

## ADD

**PARAMETERS**

    a (int) - The first integer.
    b (int) - The second integer.

**EXAMPLE USAGE**

```cpp
    int a = 10;
    int b = 20;

    cout << add(a, b) << endl;
```

Output: `30`
