# DOCUMENTATION <h><h/>

Quick guide for `uplow.h`. <br> <br>

# UPPER <h><h/>

<b> Parameters </b>
    a (string) - <br>
    The string that you want to make uppercase.

<b> Example Usage </b>

```cpp

    string a = "hi there!";

    a = upper(a);

    cout << a << endl;
```
<br>
Output: `HI THERE!`

# LOWER <h><h/>

<b> Parameters </b>
    a (string) - <br>
    The string that you want to make lowercase.

<b> Example Usage </b>

```cpp

    string a = "HI THERE!";

    a = lower(a);

    cout << a << endl;
```
<br>
Output: `hi there!`

# UPPER_CHAR <h><h/>

<b> Parameters </b>
    a (char) - <br>
    The character that you want to make uppercase.

<b> Example Usage </b>

```cpp

    char a = "a";

    a = upper_char(a);

    cout << a << endl;
```
<br>
Output: `A`
