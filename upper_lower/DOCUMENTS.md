# DOCUMENTATION

Quick guide for `uplow.h`. <br> <br>

## UPPER

**PARAMETERS**

a (string) - The string that you want to make uppercase.

**EXAMPLE USAGE**

```cpp
    string a = "hi there!";

    a = upper(a);

    cout << a << endl;
```

Output: `HI THERE!`

## LOWER

**PARAMETERS**

a (string) - The string that you want to make lowercase.

**EXAMPLE USAGE**

```cpp
    string a = "HI THERE!";

    a = lower(a);

    cout << a << endl;
```

Output: `hi there!`

## UPPER_CHAR

**PARAMETERS**

a (char) - The character that you want to make uppercase.

**EXAMPLE USAGE**

```cpp
    char a = "a";

    a = upper_char(a);

    cout << a << endl;
```

Output: `A`

## LOWER_CHAR <h><h/>

**PARAMETERS**

a (char) - The character that you want to make lowercase.

**EXAMPLE USAGE**

```cpp
    char a = "A";

    a = lower_char(a);

    cout << a << endl;
```

Output: `a`
